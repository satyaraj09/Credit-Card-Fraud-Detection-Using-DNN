Credit Card Fraud Detection using DNN

Problem Type: Binary Classification Key Challenge: Extreme class imbalance (fraud 0.17%)

Dataset: Kaggle Credit Card Fraud Detection Features are anonymized (V1V28 from PCA), plus Time, Amount, and target Class.
